# RatCar
Interface Cérebro-Máquina (ICM) inspirada em pesquisas japonesas, desenvolvida para fins recreativos. O projeto integra sinais neurais de um rato a um sistema robótico motorizado, permitindo que o movimento do robô seja guiado por atividade cerebral em tempo real. Desenvolvido por Antonio e Lucas como projeto de exploração em neuro-robótica.
